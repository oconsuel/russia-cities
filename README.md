# Датасет городов, субъектов и округов Российской Федерации

---

Датасет содержит справочные данные о федеральных округах, субъектах (регионах) и городах Российской Федерации

Датасет будет полезен разработчикам и аналитикам, которым нужен готовый справочник административно-территориального деления России. 

Его можно использовать для наполнения выпадающих списков и форм выбора региона/города, для геокодирования и нормализации адресов, для агрегации и визуализации статистики по округам и субъектам, для построения справочников в CRM и внутренних сервисах, а также в учебных и исследовательских проектах.

## 1. Актуальность данных

Данные собраны из разных источников, опубликованных в 2025–2026 годах, и актуализированы по состоянию на 2026 год. Перечень источников приведён в разделе [5. Источники данных](#5-источники-данных).

На текущий момент в датасете содержится информация о федеральных округах, субъектах федерации и городах Российской Федерации:

## 2. Общее описание файлов

| Файл | Описание |
| --- | --- |
| [`ru-districts.json`](ru-districts.json) | Федеральные округа РФ: общие характеристики округов (население, площадь, количество субъектов и городов). |
| [`ru-subjects.json`](ru-subjects.json) | Субъекты федерации (регионы) в составе федеральных округов. |

## 4. Примечания к данным файлов

- Данные актуализированы с учётом действующих указов Президента, территориальной целостности и т. п. РФ на 2026 год.
- К сожалению, в общем доступе отсутствует часть информации о новых регионах РФ, площади их территорий и численности населения, из-за чего данные по отдельным городам, субъектам и округам могут быть искажены.
- В перечень городов, в отличие от большинства открытых данных и сетевых датасетов, включены отдельными городами населённые пункты внутри Москвы / Санкт-Петербурга / Севастополя (города в составе городов федерального значения).
